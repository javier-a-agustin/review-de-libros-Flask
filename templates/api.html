Documentacion de la API

Obtener un libro por isbnumber

"/api/num/{isbnumber}" devuelve la informacion relacionada a el libro con el numero "isbnumber" especificado.

Yendo a: "/api/num/0765317508"

Lo que se recibe.

{ "author": "Gary Jennings", "average_score": 0, "isbn": "0765317508", "review_count": 0, "title": "Aztec", "year": "1980" }

Cuando se llama a la API con un numero no valido:

Llamada: "/api/num/01"

{ "error": "Error: Ese isbn no existe" }

Obtener todos los isbnumber.

Dirigiendonos a "/api/all_isbn" se obtendra:

{ "all_isbnumbers": [ { "isbn": "isbn" }, { "isbn": "0380795272" }, ... { "isbn": "0553262149" } ] }

Obtener todos los libros.

Dirigiendonos a "/api/all" se obtendra:

{ "all_books": [ { "author": "author", "isbn": "isbn", "publicationyear": "year", "title": "title" }, ... , { "author": "L.M. Montgomery", "isbn": "0553262149", "publicationyear": "1925", "title": "Emily Climbs" } ] }

Obtener todas las reviews de un libro por isbnumber

Dirigiendonos a "/api/reviews/{isbnumber}", donde isbnumber es el numero asociado al libro, se obtendra:

{ "Todas las reviews:": [ { "isbnumber": "1481438867", "reviewtext": "muy bueno", "value": 3 }, { "isbnumber": "1481438867", "reviewtext": "malardisimo", "value": 1 }, { "isbnumber": "1481438867", "reviewtext": "Me gusto bastante", "value": 3 } ] }
